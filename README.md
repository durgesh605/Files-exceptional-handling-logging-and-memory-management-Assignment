# Python Practical Assignment

##  About This Project
This folder contains **Python practical programs (Q1–Q23)** based on:
- File Handling  
- Exception Handling  
- Logging  
- Memory Management  

Each question is written in a **separate Python (.py) file**.

 **One Question = One File = One Run**

---

##  Folder Structure
```
Practical_Answer/
│
├── Q01_write_file.py
├── Q02_read_file.py
├── Q03_file_not_found.py
├── ...
├── Q23_log_file_error.py
│
├── sample.txt
├── empty.txt
``` 

 Run **only one file at a time**.

---

##  Important Files

### sample.txt (Required)
Used for file read/write questions.
```
Hello, this is a sample file.
```

### empty.txt (Optional)
Used only to show empty file handling.
